# Franvy的日志博客

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 开发

```bash
pnpm dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看博客。

### 构建

```bash
pnpm build
pnpm start
```

## 技术栈

- **框架**: Next.js 16
- **React**: 19.2.0
- **文档**: Fumadocs 16.x
- **样式**: Tailwind CSS 4
- **包管理器**: pnpm
- **运行时**: Node.js >= 18.17.0
